# Teams-Namens-Generator

## Allgemein

| Attribut            | Wert                                                                                                   |
| ------------------- | ------------------------------------------------------------------------------------------------------ |
| Systematischer Name | `teams-name-generator`                                                                                 |
| Funktionalität      | Konsequente Namen für Microsoft Teams Teams generieren                                                 |
| Stack               | Svelte, Firebase, Postcss, daisyUI, Vite                                                               |
| Deployment          | GitHub Pages                                                                                           |
| URL                 | [https://team-name.birklehof.space/](https://team-name.birklehof.space/)                               |
| Code                | [https://github.com/Birklehof/teams-name-generator](https://github.com/Birklehof/teams-name-generator) |

## Wichtige Befehle

- `yarn run dev` - Startet den Entwicklungsserver
- `yarn run build` / `yarn run predeploy` - Erstellt die Website (Ausgabe in `dist/`)
- `yarn run preview` - Vorschau der erstellten Website
- `yarn run deploy` - Veröffentlichen der erstellten Website auf GitHub Pages

## Deployment

Dieser Dienst wird über [GitHub Pages](https://pages.github.com/) bereitgestellt. Der `gh-pages`-Zweig enthält die erstellte Website (einfache HTML-, CSS- und JS-Dateien aus dem Ordner `dist/`), die über GitHub Pages bereitgestellt wird. Um die Website zu aktualisieren, musst du Folgendes tun:

1. Änderungen committen und pushen
2. `yarn run predeploy` ausführen - Dadurch wird die Website im Ordner `dist/` erstellt
3. Die erstellte Website mit `yarn run preview` überprüfen (optional, aber empfohlen)
4. `yarn run deploy` ausführen - Dadurch wird die erstellte Website auf GitHub Pages veröffentlicht
