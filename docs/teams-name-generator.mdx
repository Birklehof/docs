# Teams Name Generator

## General

| Field      | Value                                                                                                  |
| ---------- | ------------------------------------------------------------------------------------------------------ |
| Name       | `teams-name-generator`                                                                                 |
| Purpose    | Generate consistent names for Microsoft Teams                                                          |
| Stack      | Svelte, Firebase, Postcss, daisyUI, Vite                                                               |
| Deployment | GitHub Pages                                                                                           |
| URL        | [https://birklehof.github.io/teams-name-generator/](https://birklehof.github.io/teams-name-generator/) |
| Code       | [https://github.com/Birklehof/teams-name-generator](https://github.com/Birklehof/teams-name-generator) |

## Important commands

- `yarn run dev` - Run the development server
- `yarn run build` / `yarn run predeploy` - Build the website (output in `dist/`)
- `yarn run preview` - Preview the built website
- `yarn run deploy` - Deploy the built website to GitHub Pages

## Deployment

This service is deployed via [GitHub Pages](https://pages.github.com/). The `gh-pages` branch of the repository contains the built website (plain HTML, CSS and JS files from the `dist/` folder), which is served by GitHub Pages. To edit the website follow these steps:

1. Commit and push your changes
2. Run `yarn run predeploy` - This will build the website in the `dist/` folder
3. Preview the built website with `yarn run preview` (optional, but recommended)
4. Run `yarn run deploy` - This will deploy the built website to GitHub Pages
